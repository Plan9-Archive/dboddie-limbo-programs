Limbo Examples
==============

drawtest.b          Draws on an image and saves the result to a PPM image, or
                    to stdout if no file name is passed as an argument.

drawtest-wm.b       Draws on an image and saves the result to a PPM image, or
                    to stdout if no file name is passed as an argument.
                    Requires the window manager to be running.

inherit.b           Demonstrates a simple form of inheritance by wrapping a
                    base implementation of an ADT in a more specialised ADT and
                    forwarding method calls to the base implementation before
                    performing specialised tasks.
